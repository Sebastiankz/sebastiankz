### BQ-SafeRoutes
**Geospatial road-safety platform for Barranquilla**

Crowdsourced incident reports feed a DBSCAN-based clustering pipeline that surfaces accident hotspots. FastAPI and PostGIS power the analytics layer; a React dashboard and a React Native (Expo) app handle visualization and field reporting.

`FastAPI` `PostgreSQL` `PostGIS` `React` `React Native` `Docker`

### La Cuenta
**Local-first POS for a real family business**

Event-sourced backend with an outbox pattern, built so the app keeps working — and stays in sync — on unreliable connectivity. Designed around a real constraint: payment is verbal, so the app has to win on usability to actually become the source of truth.

`FastAPI` `PostgreSQL` `Event Sourcing` `PWA`
